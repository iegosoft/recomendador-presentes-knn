Sobre o Sistema

Este sistema aplica conceitos de álgebra linear para recomendar presentes personalizados. Cada item do catálogo e o perfil do destinatário são representados como vetores em um espaço de 19 dimensões (multi-hot encoding), onde cada dimensão corresponde a um interesse (tecnologia, música, viagem, etc.).

O algoritmo K-Vizinhos Mais Próximos (KNN) usa a similaridade de cosseno — métrica baseada no ângulo entre vetores, um conceito central de álgebra linear — para encontrar os itens mais compatíveis com o perfil informado. A compatibilidade exibida é calculada pelo Coeficiente de Sobreposição, que mede a interseção entre os conjuntos de tags do perfil e do item.
